# 12306 网站余票查询脚本

12306火车票余票查询实时发送邮件及时提醒是否有票！

![image](https://github.com/user-attachments/assets/dace2c02-6908-45ca-87a5-6baa9d0db589)

![image](https://github.com/user-attachments/assets/2a2cdd74-cb62-4f4c-99be-6ad717862d98)
